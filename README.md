# 🌤 Weather Dashboard

A simple Weather Dashboard web application developed using HTML, CSS, and JavaScript. The project includes user authentication using a local JSON file and displays live weather information using an Open API.

---

## 📌 Project Overview

This project demonstrates:

- Semantic HTML
- CSS Styling
- JavaScript DOM Manipulation
- Fetch API
- JSON Authentication
- Open Weather API
- Git & GitHub Version Control

---

## ✨ Features

### 🔐 Login Authentication

- Login using Username and Password
- Credentials fetched from `users.json`
- Authentication using JavaScript Fetch API
- Redirects to Dashboard after successful login

---

### 🌍 Weather Dashboard

- Search weather by city
- Displays:
  - 🌡 Temperature
  - 💧 Humidity
  - 💨 Wind Speed
  - ☁ Weather Condition

---

### 📅 Weather Forecast

- Displays weather forecast for upcoming days
- Dynamic weather information using API

---

### 🎨 Responsive Design

- Mobile Friendly
- Tablet Friendly
- Desktop Friendly

---

### 🚪 Logout

- Logout button returns user to Login page
